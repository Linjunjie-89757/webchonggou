## 验收重点

1. 页面没有火星文、明显英文泄漏或不一致按钮文案。
2. 运行、调试、批量运行期间按钮有 loading 或禁用保护，避免重复提交。
3. 失败报告能看清步骤、类型、定位器、输入/目标、错误信息和截图。
4. 批次报告能快速判断哪条用例失败，并能跳到单条运行报告。
5. 环境和变量集继承规则在运行弹窗、环境列表、报告上下文中都能解释清楚。
6. CI 返回的 `reportUrl` 能作为流水线日志回跳入口。
