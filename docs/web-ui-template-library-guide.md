# Web UI 模板库使用说明

## 定位

Web UI 模板库用于把稳定的登录、查询、表单、审批等自动化流程沉淀为团队可复用模板。模板按工作空间隔离，和 Web UI 用例并列维护。

## 使用入口

- 在 Web UI 自动化页面点击“从模板新建”，进入模板库弹窗。
- 页面会优先加载当前工作空间的团队模板。
- 如果模板库为空或接口暂不可用，会显示内置模板兜底，不影响继续创建用例。
- 在用例列表点击“保存为模板”，可以把已有用例沉淀为模板。

## 推荐流程

1. 先创建并调试一条稳定的 Web UI 用例。
2. 在用例列表点击“保存为模板”。
3. 输入模板名称并保存。
4. 同事通过“从模板新建”选择该模板。
5. 编辑器打开后按目标系统微调地址、定位器和变量，再保存为新用例。

## 当前能力

- 查询工作空间模板列表。
- 使用模板创建新用例草稿。
- 从已有用例保存为模板。
- 编辑模板基础信息。
- 删除模板。
- 模板库为空时显示内置模板兜底。

## 当前边界

- 第一版模板编辑主要维护名称、模块、描述、浏览器、超时、状态等基础信息。
- 模板步骤建议通过“已有用例保存为模板”沉淀。
- 暂不做模板版本管理、审核发布、跨工作空间共享市场和 AI 生成模板。
